What you can do

Compare through time

Two analytes on a shared time axis — normalized to compare shape honestly, or raw on dual axes with an explicit caveat.

See the seasonal pattern

Monthly climatology and a real STL decomposition of the measured record — trend and seasonal cycle pulled from the actual data, with no fabricated forecasts.

Predict an analyte

A model that estimates the main analyte from its three best-correlated analytes, scored against a mean-only baseline so you can see whether it actually adds anything.

Test the relationship

Least-squares fit with a confidence band, R²/p/n, and a temporal-autocorrelation flag so the p-value isn't oversold.

Screen correlations

Main analyte vs every other — Spearman by default, n shown per row, low-n flagged, multiple-comparisons caveat printed.

Place it on the map

All 34 NEON aquatic sites, from desert streams to Alaskan lakes to Puerto Rican rivers.

Statistics

The analysis is built on transparent, well-documented methods:

  • Relationships use ordinary least squares with R², p and n, flagged for temporal autocorrelation; correlations default to Spearman (robust to skewed chemistry) with an n ≥ 8 reliability guard.
  • Seasonality is a real STL decomposition, with interpolated months disclosed, rather than a fitted sine wave.
  • Every statistic reports its sample size, and below-detection values are flagged rather than hidden (15,553 across the record).
  • Analyte names and units are verified (bromide vs bicarbonate, chloride vs chlorine, ANC in meq/L).
